# servicex-sdk-rn

A serviceX sdk for react-native

## Installation

```sh
npm install servicex-sdk-rn
```

## Usage

```js
import ServicexSdkRn from "servicex-sdk-rn";

// ...

const result = await ServicexSdkRn.multiply(3, 7);
```
